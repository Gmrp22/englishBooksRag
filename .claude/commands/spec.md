# User Input
`$ARGUMENT` is the topic you want to learn

Example:
- "Angular 20"
- "I want to learn basic backend architecure principles"

# Context
Generate a new plan for learning based on the user input, the output must be a markdwon file under `docs/learn/<topic>.md`.

Example
`docs/learn/angular-20.md`

Make sure to be clear and extensive when needed.
